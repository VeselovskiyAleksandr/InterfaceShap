#InterfaceShap
InterfaceShap - кросплатформенная программа для нахождению площади фигур
и размера описываемого вокруг них прямоугольника по параметрам, вводимым ипользователем.
Классы фигур наследуются от родительского класса Shape.
Программа проверяет правильность ввода для всех фигур на соответствие значений параметров 
больше нуля. Кроие того, проводится проверка ввода значений параметров треугольника на 
соответствие условию: сумма двух любых сторон больше третьей стороны.
Для каждой фигуры создаётся  структура BoundingBoxDimensions, которая содержит размеры 
описываемого прямоугольника.
Функция вывода результатов помещена в отдельный файл printParams.h

##Установка                                                                                                                                                                                                            
 Клонируйте репозиторий:
   bash git clone https://github.com/VeselovskiyAleksandr/InterfaceShap.git
   cd worldCinemaInfoSearchEngine   
   Создайте директорию для сборки:
   bash
   mkdir build
   cd build                                                                                                                                                                                                            
   Запустите CMake:
   bash
   cmake ..                                                                                                                                                                                                            
   Соберите проект:
   bash
   cmake --build .                                                                                                                                                                                                     
   Запустите исполняемый файл:
   bash
   ./worldCinemaInfoSearchEngine
